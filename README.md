# ceasercipher
Collection of various substitutions techniques that all play on the Ceaser substitution method

### Glossary
C = Cipher or encrypted letter

K = Keyword letter

P = Plain letter

S = Sum

### Substitution methods
#### Ceaser: 
  
Pi + Sub = Ci

Ci - Sub = Pi
  
#### Keyword:

Pi + Ki = Ci
  
Ci - Ki = Pi
  
#### Keyword Sum:

Pi + Ki + Si-1 = Ci

Ci - Ki - Si-1 = Pi

Si = Si-1 + Ci

S0 = initial value

S1 = S0+C1,  S2 = S1+C2, ...
